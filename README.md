# JUNO

JUNO (Java Unified Native Operation) is a lightweight Java framework to write microcontroller programs using familiar `setup()` and `loop()` methods. Currently it targets ESP32 with plans to support other MCUs.

## Features

- Write embedded code in Java  
- Supports ESP32  
- Simple CLI tools for flashing and monitoring
